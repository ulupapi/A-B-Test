# A/B Test Demo

Интерактивная демонстрационная страница для защиты учебной работы.

## Что внутри

- аккуратная структура с разделами, таблицами и графиками;
- страница подходит для показа преподавателю в браузере;
- есть онлайн-версия и локальная версия для презентации.

## Ссылки

- Онлайн-страница: https://ulupapi.github.io/A-B-Test/
- Репозиторий: https://github.com/ulupapi/A-B-Test

## Как показать преподавателю

1. Откройте онлайн-страницу по ссылке выше.
2. Если изменения не видны сразу, обновите страницу без кэша.
3. Для офлайн-показа используйте локальную версию на своем ноутбуке.
