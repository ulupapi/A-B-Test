Стратегия A/B тестирования алгоритма рекомендаций

Веб-приложение на Streamlit по теории игр: доминирование, седловая точка, графоаналитический метод, симплекс-метод и Браун–Робинсон.

Исходная матрица A1: [9, 17, 7], A2: [15, 7, 13]
Результат Цена игры: V = 10.75
Оптимум A p1 = 0.375, p2 = 0.625
Оптимум B q1 = 0, q2 = 0.375, q3 = 0.625

Если приложение запускается локально, откройте http://localhost:8501.
